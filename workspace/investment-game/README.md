# Investment Game

This directory contains the tablet app and services require to conduct an investment game experiment.

### Getting Started

To run the full system, follow the setup instructions in the respective subdirectories:

- Frontend (Tablet App): See the [README](./frontend/README.md) in `./frontend`
- Backend Services: See the [README](./services/README.md) in `./services`

Please make sure both the frontend and services are properly configured and running for the game to function as intended. Alternatively, do whatever the fuck you want \\ (•◡•) \/
